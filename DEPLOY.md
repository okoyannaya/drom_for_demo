# Инструкция по деплою на Netlify

## Способ 1: Через Netlify CLI

1. Установите Netlify CLI глобально:
```bash
npm install -g netlify-cli
```

2. Авторизуйтесь в Netlify:
```bash
netlify login
```

3. Соберите проект:
```bash
npm run build
```

4. Задеплойте проект:
```bash
netlify deploy --prod
```

## Способ 2: Через GitHub и Netlify Dashboard

1. Загрузите проект в GitHub репозиторий

2. Зайдите на [netlify.com](https://www.netlify.com) и войдите в аккаунт

3. Нажмите "Add new site" → "Import an existing project"

4. Выберите ваш GitHub репозиторий

5. Настройки билда:
   - **Build command:** `npm run build`
   - **Publish directory:** `out`

6. Нажмите "Deploy site"

## Способ 3: Через drag & drop

1. Соберите проект локально:
```bash
npm install
npm run build
```

2. Зайдите на [app.netlify.com/drop](https://app.netlify.com/drop)

3. Перетащите папку `out` на страницу

## Тестирование статического экспорта локально

Перед деплоем на Netlify рекомендуется протестировать статический экспорт локально:

1. Соберите проект:
```bash
npm run build
```

2. Запустите локальный статический сервер:
```bash
npm run serve
```

Или используйте один из альтернативных способов:

### Вариант 1: С помощью serve (уже добавлен в скрипты)
```bash
npm run serve
```

### Вариант 2: С помощью Python (если установлен)
```bash
cd out
python -m http.server 3000
```

### Вариант 3: С помощью Node.js http-server
```bash
npx http-server out -p 3000
```

### Вариант 4: С помощью PHP (если установлен)
```bash
cd out
php -S localhost:3000
```

После запуска любого из способов откройте в браузере:
- `http://localhost:3000` - главная страница
- `http://localhost:3000/listing/627037398/` - страница просмотра объявления

**Важно:** Обязательно проверьте:
- ✅ Все чекбоксы работают
- ✅ Все кнопки реагируют на клики
- ✅ Форма отправляется корректно
- ✅ Навигация между страницами работает
- ✅ localStorage сохраняет данные
- ✅ Нет ошибок в консоли браузера (F12)

## Проверка после деплоя

После деплоя ваше приложение будет доступно по адресу вида:
`https://your-app-name.netlify.app`

## Примечания

- Проект настроен для статического экспорта
- Все данные хранятся в localStorage браузера
- Динамические маршруты будут работать через статические файлы
- При обновлении страницы на `/listing/[id]` Netlify автоматически перенаправит на index.html
