# Drom Car Listing Application

Приложение для размещения объявлений о продаже автомобилей на платформе Дром.

## Технологии

- Next.js 14
- React 18
- TypeScript
- CSS Modules

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Страницы

- `/` - Форма размещения объявления о продаже автомобиля
- `/listing/[id]` - Страница просмотра объявления

## Функциональность

### Форма размещения объявления

- Базовые поля автомобиля (VIN, марка, модель, год, пробег)
- Выбор количества владельцев
- Выбор цвета через цветные кнопки
- Особые отметки с условным показом полей и предупреждений
- Описание
- Цена и валюта
- Возможность обмена с дополнительными опциями
- Статус продажи
- Город и регион
- Контактная информация
- Цифровой ассистент с дополнительными полями

## Сборка для продакшена

```bash
npm run build
npm start
```
